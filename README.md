## 📖 Overview

This project is a fully dynamic and responsive **Portfolio Website** developed using **Django**. It allows complete portfolio management through the Django Admin Panel without modifying the source code.

Projects, Skills, Resume, and Contact Messages are dynamically managed from the admin dashboard, making the portfolio easy to maintain and update.

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### Navigate

```bash
cd your-repository
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```
